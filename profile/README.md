The **Aptos Coding Challenge** is a beginner-friendly series where developers work through simple projects containing intentional issues or incomplete components. Participants fork the repository, analyze the codebase, and submit a pull request with their solution or implementation.

By completing these exercises, you'll gradually strengthen your Move development skills and gain practical experience to build your own  project on Aptos!
